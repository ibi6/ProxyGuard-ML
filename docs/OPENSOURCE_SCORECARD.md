## 当前等级

**A− · 成熟的研究 / 演示级开源项目**

## 与「优秀生产级开源」的差距

| 差距 | 严重度 |
|------|--------|
| 无多用户登录 / SSO | 对 SaaS 高，对研究演示可接受 |
| 默认同数据 | 若宣称「检测产品」则高 |
| 进程内训练线程 | 中 |
| 前端依赖 CDN（Tailwind/Chart） | 中（CSP 已说明） |
| 无自动发版 / 上 PyPI | 低 |

## 若宣称企业产品则必须补

1. 真实鉴权与审计  
2. 真实流量特征管线  
3. 训练进程隔离  
4. 密钥与配置管理  

## 仍可做的增强（研究级）

1. 静态资源本地化  
2. pre-commit  
3. 覆盖率徽章  
4. 打 tag 自动 Release
