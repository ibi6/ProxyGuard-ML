## 第三阶段：数据库优化

### 设计定位（答辩口径）

SQLite 只存 **训练任务、预测日志、设置 KV**，业务样本在 CSV，模型在 joblib。  
这是**实验控制台存储**，不是 IPAM 业务库，不要按 3NF 企业 ERP 标准硬套。

### 完整 SQL

见仓库：`docs/schema.sql`（与 `app/db.py` 一致，含索引）。

### 优点

- 主键明确；任务与日志分离  
- 设置 JSON KV 扩展方便  
- 已补 status / 时间索引  

### 后续可选（加分项，非必须）
